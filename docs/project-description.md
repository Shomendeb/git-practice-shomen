# Project Technical Overview

This project is a Python-based practice application demonstrating architectural layout and version control. 

## Technical Stack
- **Language:** Python
- **Tools:** Git, GitHub, VS Code

## Directory Structure
- `src/`: Houses the functional code modules including core utilities and the main entry point execution script.
- `docs/`: Holds project documentation and technical references.

## Project Changelog
- **v1.0.0**: Initial structural setup and basic script execution.
- **v1.1.0**: Implemented arithmetic utilities (addition, subtraction, multiplication).
- **v1.2.0**: Integrated robust division functionality with zero-division error tracking.
